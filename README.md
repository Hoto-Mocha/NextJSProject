## 무엇을 배웠나요?
웹 개발 최신 기술인 NextJS를 개인 프로젝트에서 활용하면서 NextJS의 사용법을 공부하고, 요구사항 분석과 기능 정의, 자료사전 등을 만드는 요구사항명세서 작성에 대해 배웠습니다.
